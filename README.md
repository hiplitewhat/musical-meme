# HTML5 Obby Game

Pure HTML/CSS/JS canvas-based obstacle course (obby) with multiple levels, checkpoints, hazards, and best-time tracking.

## How to play

Open `index.html` in a modern browser.

### Controls

- Move: **A/D** or **Left/Right**
- Jump: **Space** / **W** / **Up**
- Respawn (costs a life): **R**

### Features

- 3 levels with progressively harder obstacle layouts
- Checkpoints (respawn at last reached checkpoint)
- Hazards: spikes, rotating sweepers, falling blocks, moving platforms
- Level completion + unlock next level
- Best times per level stored in `localStorage`
